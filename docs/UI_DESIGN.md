# DiyyMusic UI Design

The v0.7.0 interface follows the approved pink-and-white premium design:

- Official pink music-note branding
- Light canvas with restrained pink accents
- Rounded modern media cards
- Liquid Glass only where depth is useful
- Custom-looking playback controls
- Clean sliders without vertical handles
- Home, Search, Library, and Profile primary navigation
- Profile contains a dedicated Account & Token flow
